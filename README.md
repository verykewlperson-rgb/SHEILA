# Hindi Seekhen

Hindi Seekhen is a Perplexity-inspired, AI-powered language lab designed for serious multilingual learners. The experience trades games for precision: a sidebar-first dashboard, neutral palette, and card-based layout keep the focus on conversation practice, pronunciation drills, mistake review, and competitive dueling across languages such as Hindi, English, Spanish, French, Mandarin, and more. 

## Vision & Core Design Principles
- **Formal + Minimal:** Clean typography, squared geometry, subtle borders, and restrained motion deliver a premium professional feel. 
- **Neutral Palette:** Light and dark themes rely on white/off-white, greys, and deep navy/black shades rather than saturated or playful colors. 
- **Task-focused Layout:** Persistent sidebar navigation, a compact top bar, and modular cards make the interface scalable as new AI-powered modules are added.
- **Multi-language Ready:** Every motion and copy reflects a platform where Hindi, English, Spanish, and future languages sit side by side.

## Layout Overview
1. **Sidebar:** Vertical, icon + label navigation with the app logo `हिंदी सीखें`, a Bronze | 10 pts 🇮🇳 badge at the top, navigation for the core modules, and Settings/Logout at the bottom.
2. **Top Bar:** Search input, language selector, theme toggle, and profile/avatar dropdown.
3. **Workspace:** Each page (Home, Live Speaker, Mistakes, Dueling, Hear & Type, Pronouncer, Profile) uses edge-to-edge panels, stat cards, and clean data grids that emphasize clarity and usability.
4. **Authentication:** Login/Signup live in centered cards with options for Google, Apple, and email sign-in while preserving the same typography and border-focused aesthetic.

